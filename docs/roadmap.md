# Roadmap

## MVP

- FastAPI backend
- SQLite local persistence
- Search profiles and resumes
- Greenhouse and Lever connectors
- Manual URL import
- Dedupe and provenance
- Ollama scoring with fallback
- Dashboard and job detail UI
- Scheduler and sync logs

## Phase 2

- Better company page toolkit
- Import/export workflows
- Enhanced analytics
- Playwright-assisted helpers where explicitly needed
- Better source-specific onboarding
- Packaging and release polish

## Known limitations

- Only Greenhouse and Lever are live connectors in this MVP.
- Experimental connectors are honest scaffolds, not full scraping promises.
- The frontend is intentionally minimal and prioritizes data visibility over advanced interaction polish.
