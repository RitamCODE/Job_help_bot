# Contributing

Thanks for helping improve OpenJobs Local.

## Development flow

1. Create a feature branch.
2. Set up the backend and frontend locally.
3. Add or update tests for core behavior.
4. Document source limitations honestly when adding connectors.
5. Keep the project local-first and avoid paid API assumptions.

## Pull request expectations

- Explain the user-facing change.
- Call out connector caveats or restrictions.
- Include screenshots for UI work when practical.
- Note any Ollama-specific assumptions.
